Scroll Animation Demo

Step 1

The first step involves setting up the website. Create the default HTML, JS, and CSS files and link them together. Then fill the page with divs so that you can scroll up and down and view the different content. Use Lorem to add text and add a reveal class to each div. You can add multiple of the divs in the sample code below to fill the page.

div class="container one reveal"

Step 2

Time to add the CSS. Go to the CSS and style the page as wanted. Add a style to the .reveal class in css. Give it a position of relative and position off the screen to the top. Make its opacity 0 (the user can't see it yet) and give it a transition of 2s. Also, add a class of .active to the stylesheet and give it the properties to position it to start at Y = 0. Give it an opacity of 1.

.reveal{
position: relative;
transform: translateY(150px);
opacity: 0;
transition: 2s all ease;
}
.active{
transform: translateY(0);
opacity: 1;
}

Step 3

Time to add the JS. To create the function we will need to check it each time a user scrolls (either up or down), check the users location on a page, and see where each div is located.

function reveal() {
var reveals = document.getElementsByClassName("reveal");
for (var i = 0; i < reveals.length; i++) {
var windowHeight = window.innerHeight;
var elementTop = reveals[i].getBoundingClientRect().top;
var elementVisible = 150;
if (elementTop < windowHeight - elementVisible) {
reveals[i].classList.add("active");
}
}
}
window.addEventListener("scroll", reveal);
// To check the scroll position on page load
reveal();

Now, lets break it down. We are creating a function called reveal that we will call each time the user "scrolls". Then, we are getting all the divs with class reveal using document.getElementsByClassName("reveal"). For each element we go through one by one and check to see the windowHeight (how far the user has scrolled), get the divs distance from the top, and setting a variable elementVisible to set the distance at which the div should become visible. Then we check if the element is visible and if it is, we add the active class to the div. This will call the transition the active class contains and move the div into view.

What are Pseudo Classes?

Again, if we think of CSS in terms of spv, then psuedo class is an add-on to a selector.

            selector: pseudo_class {property: value;}

We use them for links

If we want our link to look different, we can simply add the following

            a {color: red;}

but if we want each of our link states to look different we will use a psuedo class

  • :hover
  • :active
  • :visited
